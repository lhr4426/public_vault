# 스플래시 이미지

- Edit > Project Settings > Player > Splash Image
- Animation : Dolly - 잠깐 커짐 / Static - 일정 크기


# Mesh Type

- Full Rect : 투명한 부분을 포함하는 전체 사각형으로 렌더링
- Tight : 투명한 부분을 제외하고 렌더링

# 소리 재생하기

- 소리를 재생하는 객체인 AudioSource와 재생할 소리인 AudioClip이 필요
- AudioSource.PlayOneShot(clip) : 오디오끼리 겹치지 않고 한 번만 소리를 재생함
- AudioSource.Play(clip) : 계속 재생함 (BGM 등)
- BGM 등을 씬 전환 후에도 계속 틀기 위해서는 DontDestroyOnLoad를 사용하면 됨

# 빌드 설정

- Resolution and Presentation 에서, Landscape = 가로모드, Portrait = 세로모드 를 의미함
- 안드로이드 마켓에 배포하기 위해서는 64bit 지원이 필수. Scripting Backend를 IL2CPP로 변경하고 Target Architectures에서 ARM64를 체크해야 함.
- Publishing Settings에서, Keystore를 추가해야 함.
