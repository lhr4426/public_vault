# Canvas의 RenderMode 차이

- Overlay : 스크린 위에 직접 그려짐
- World Space : 게임 World에 그려짐

# Pivot

- x : 0.5, y : 0.5면 기준점이 가운데라는 의미
- 만약, HP바를 만들려면 x : 0으로 함으로써 기준점을 왼쪽으로 잡는 것이 좋다.

# RigidBody를 쓰되 중력을 적용하지 않는 방법

- Body Type을 Kinematic으로 바꾸면 물리가 적용되지 않는다.
	- 대신 Collider의 IsTrigger를 체크해줘야 한다.
	- 그리고 함수를  OnCollisionEnter2D 말고 OnTriggerEnter2D를 사용해야 한다.
