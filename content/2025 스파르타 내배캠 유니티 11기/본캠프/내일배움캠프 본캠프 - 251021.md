## Projectile이 삭제될 때 특정 Action을 실행하도록 변경


- 기존 코드는 사라질 때 특정 코드를 실행시킬 수 없었음.
- 근데 나는 projectile이 삭제될 때 소리를 내야 함
- 그래서 Action을 인자로 받도록 변경함

```csharp
public void RequestRelease(float fallbackSeconds = 0.15f, Action callback = null)  
{  
    if (isReleasing) return;  
    OnPrepareRelease();  
    StartCoroutine(ReleaseAfterCurrentState(fallbackSeconds, callback));  
}
```
