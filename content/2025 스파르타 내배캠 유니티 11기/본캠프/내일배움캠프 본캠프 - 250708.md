### 델리게이트

- 메소드를 참조하는 타입
- == 함수 포인터
- 델리게이트를 이용하면 메소드 자체를 매개변수로 전달하거나 변수로 할당할 수 있음
- 함수를 변수로 저장할 수 있다고 생각하기

```csharp
// 예시
delegate int Calculate(int x, int y);

static int Add(int x, int y)
{
	return x + y;
}

class Program
{
	static void Main()
	{
		Calculate cal = Add;
		int result = cal(3, 5);
		Console.WriteLine("결과 : " + result);
	}
}
```
