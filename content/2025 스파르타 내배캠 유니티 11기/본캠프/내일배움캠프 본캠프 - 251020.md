# 오늘 학습 한 내용을 나만의 언어로 정리하기 

## itch.io에 빌드 올리기

1. itch.io 계정을 생성한다.
2. 우측 상단에 있는 프로필 옆 아래방향 화살표를 눌러서 대시보드로 이동  
  
![](../../image/Pasted%20image%2020251020165215.png)

3. Create a New Project 클릭
4. 타이틀, 프로젝트 URL 등등 설정
5. WebGL로 올릴거라면 Kind of Project를 HTML로 변경
	- 이 때, 게임 빌드 파일을 압축해서 올려야 하는데, 최상단에 index.html이 포함되도록 압축해야 함.
6. Release Status를 지정하고 가격을 정함
7. Zip파일을 업로드 함
8. 가로세로 크기를 맞추셈
9. 저장하면 끝!
