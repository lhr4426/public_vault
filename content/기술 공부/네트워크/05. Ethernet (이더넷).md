# CSMA/CD

- CS : Carrier Sense. 반송파 감지
    
- MA : Multi Access. 다중 접속
    
- CD : Collision Detect. 충돌 감지
    
- 자세한 내용은 아래 링크 참고
    - [위키백과](https://ko.wikipedia.org/wiki/%EB%B0%98%EC%86%A1%ED%8C%8C_%EA%B0%90%EC%A7%80_%EB%8B%A4%EC%A4%91_%EC%A0%91%EC%86%8D_%EB%B0%8F_%EC%B6%A9%EB%8F%8C_%ED%83%90%EC%A7%80)
- 비슷하지만 다른 CSMA/CA (Collision Avoid) 도 있음

## UTP

![](../../image/Pasted%20image%2020260127100631.png)

- [이미지 출처](https://cabletimetech.com/ko-kr/blogs/knowledge/the-complete-guide-to-ethernet-cables-what-you-should-know?srsltid=AfmBOooFFAiJ-Z7SvLmSo6IJ_Hip7VvjWY831GI6XUfi7DIYo0HT0WbN)
- UTP : Unshielded Twisted-Pair. 두 가닥씩 꼬여있음
- FTP : Foil screened Twisted-Pair. 두 가닥씩 꼬인 선들이 하나의 호일로 감싸짐
- STP : Shielded Twisted-Pair. 두 가닥씩 꼬인 페어 하나하나가 감싸짐
- SFTP : Shielded Foil screened Twisted-Pair. 두 가닥씩 꼬인 페어 하나하나도 감싸고, 전체를 포일 하나로도 감쌈

# 프레임 구조


![](../../image/Pasted%20image%2020260127100813.png)

- [이미지 출처](https://ko.wikipedia.org/wiki/%EC%9D%B4%EB%8D%94%EB%84%B7_%ED%94%84%EB%A0%88%EC%9E%84)

- 프리앰블 : 0과 1을 번갈아서 반복하는 바이트. 송수신간의 속도 동기화가 목적
- 프레임 시작 구분자 (SFD) : 프리앰블의 비트 패턴을 깨고 실제 프레임의 시작을 알림
- MAC 목적지 (DM)
- MAC 원본 (SM)
- 이더타입 또는 길이 : 10진수로 1536, 16진수로 0x600 이상이면 이더타입, 이하면 길이를 의미함
- 페이로드
- FCS (Frame Check Sequence) : 수신측 오류 검출을 위한 CRC 코드
