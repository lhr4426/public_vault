## 1. 네트워크 만들기

- 컨테이너는 기본적으로 아무 설정을 하지 않으면 다른 컨테이너와 통신할 수 없음
- 여러 컨테이너를 하나의 네트워크로 묶는다면 가능함


```bash
docker network create <네트워크명>
```


- 예시

```bash
docker network create todo-app
```
