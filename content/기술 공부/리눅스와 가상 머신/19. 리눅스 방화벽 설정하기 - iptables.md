# iptables?

- NetFilter한테 규칙을 적용시키는 도구

# 0. 백업하기


```bash
iptables-save <위치>
```

- iptables의 정책들을 위치에 저장함

- 예시

```bash
iptables-save /data/main/backup/iptables.backup
```


# 1. 확인하기


```bash
iptables -L
```

  
![](../../image/스크린샷%202026-01-23%20오후%205.03.30.png)

- 뭔가 이것저것 많이 올라옴


```bash
iptables -t <테이블명> -L 
```

- 특정 테이블이 적용된 체인만 나옴

# 2. 수정하기

## 1) 화이트리스트 방식으로 만들기

- 일단 기존 정책 싹 다 지움


```bash
iptables -F
iptables -X
```

- 확인해보면 훨씬 정책이 줄어든걸 볼 수 있음

![](../../image/스크린샷%202026-01-23%20오후%205.08.17.png)  

- 이걸 이제 들어오는걸 막고 나가는거만 허용할거임
