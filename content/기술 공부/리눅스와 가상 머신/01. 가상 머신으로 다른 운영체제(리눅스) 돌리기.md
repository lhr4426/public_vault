# 가상 머신?

- 컴퓨터 시스템을 가상화 한 것
- 컴퓨터 한 대에 여러 운영체제를 심을 수 있음

# 사용법

- 가상머신을 돌리게 해주는 소프트웨어는 다양하게 있음
	- VM ware, VirtualBox, ...
- 맥북은 UTM 이라고 호환 잘되는거 있음
- 근데 다른 소프트웨어도 사용법은 비슷함


## 0. 원하는 운영체제의 디스크 파일 (iso)을 구해두기

- 여기서는 CentOS 7이랑 Rocky Linux 9를 사용할 예정
- 검색하면 다 나옴

## 1. 소프트웨어 켜서 새 가상머신 만들기

- UTM 기준
![](../../image/스크린샷%202026-01-16%20오후%201.13.20.png)  
- 참고 : 
	- Rocky는 최근까지 업데이트 되다보니 맥 실리콘으로도 돌아감. 그래서 가상화가 됨
	- 근데 CentOS는 옛날 거라서 맥으로 안됨. 가상화 말고 에뮬레이션으로 해야함


## 2. 나머지는 대부분 기본값으로 지정하면 됨

- iso 파일 선택에서 아까 0번에서 다운한 이미지 파일 고르면 됨

## 3. 최초 부팅에서 OS 깔기

- 처음에 iso 파일 선택하면 그게 usb처럼 꽂혀있는걸로 세팅됨
- 그래서 그 상태 그대로 키면 OS 설치 창이 나옴
- 그대로 따라가면 됨

## 4. OS 깔고 iso 빼기

- OS 다 깔리면 한 번 끔
- 설정 들어가면 드라이브들이 보임 (아래 사진은 이미 iso 뺀 뒤라서 하나밖에 안보임)  
![](../../image/스크린샷%202026-01-16%20오후%201.16.12.png)  

- 그러면 iso 들어가있는 드라이브를 클릭해서 제거 누르면 됨
- 끝! 이 뒤부터는 그냥 키면 알아서 부팅됨
