# DAS

- Direct Attached Storage
- 하드디스크 같이 직접 컴퓨터에 붙여서 쓰는 것
- 블록 기반 스토리지

# NAS

- Network Attached Storage
- 네트워크로 접속해서 사용할 수 있는 파일 기반 스토리지
- 파일 공유가 큰 목적
- IP로 통신

# SAN

- Storage Area Network
- 블록 기반 스토리지
- FC Channel (광케이블)을 사용
