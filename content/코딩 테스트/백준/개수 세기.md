[문제 링크](https://www.acmicpc.net/problem/10807)  
난이도 : 브론즈 5

# 문제 내용

총 N개의 정수가 주어졌을 때, 정수 v가 몇 개인지 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 정수의 개수 N(1 ≤ N ≤ 100)이 주어진다. 둘째 줄에는 정수가 공백으로 구분되어져있다. 셋째 줄에는 찾으려고 하는 정수 v가 주어진다. 입력으로 주어지는 정수와 v는 -100보다 크거나 같으며, 100보다 작거나 같다.

## 출력

첫째 줄에 입력으로 주어진 N개의 정수 중에 v가 몇 개인지 출력한다.

# 문제 분석

1. 첫째 줄에 입력된 정수 크기만큼의 배열 생성하기 + 결과로 반환될 값인 result 변수 생성하기
2. 둘째 줄에 입력되는 정수들을 배열에 입력하기
3. 배열을 순회하며 셋째 줄에 입력된 정수와 같은 정수를 발견했을 시 result += 1

주어진 크기의 배열을 한 번 순회하면 되기 때문에 시간복잡도는 O(n)

# 작성한 코드

```cpp
#include <iostream>

using namespace std;

int main() {
    int n; // 입력되는 데이터 개수
    cin >> n;

    int *numList = new int[n]; // 정수 배열
    for(int i = 0 ; i < n; i++) {
        cin >> numList[i];
    }

    int m; // 찾고자 하는 값
    cin >> m;

    int result = 0;

    for(int i = 0 ; i < n; i++) {
        if(numList[i] == m) {
            result += 1;
        }
    }

    cout << result;
}
```
