# 문제 내용

첫째 줄에는 별 1개, 둘째 줄에는 별 2개, N번째 줄에는 별 N개를 찍는 문제

하지만, 오른쪽을 기준으로 정렬한 별(예제 참고)을 출력하시오.

## 입력

첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

## 출력

첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.

# 문제 분석

주어진 수 - 현재 줄 순번 만큼 띈 다음에 별 찍어야함

# 작성한 코드

```cpp
// 250109

#include <stdio.h>

int main() {
    int n, i, j;
    scanf("%d", &n);
    for (i = 1; i <= n; i++, puts("")){
        for (j = n; j > 0; j--){
            if(i >= j) printf("*");
            else printf(" ");
        }
    }
}
```
