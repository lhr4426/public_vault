# 문제 내용

다음은 식품공장의 정보를 담은 `FOOD_FACTORY` 테이블입니다. `FOOD_FACTORY` 테이블은 다음과 같으며 `FACTORY_ID`, `FACTORY_NAME`, `ADDRESS`, `TLNO`는 각각 공장 ID, 공장 이름, 주소, 전화번호를 의미합니다.

|Column name|Type|Nullable|
|---|---|---|
|FACTORY_ID|VARCHAR(10)|FALSE|
|FACTORY_NAME|VARCHAR(50)|FALSE|
|ADDRESS|VARCHAR(100)|FALSE|
|TLNO|VARCHAR(20)|TRUE|

---

### 문제

`FOOD_FACTORY` 테이블에서 강원도에 위치한 식품공장의 공장 ID, 공장 이름, 주소를 조회하는 SQL문을 작성해주세요. 이때 결과는 공장 ID를 기준으로 오름차순 정렬해주세요.

# 문제 분석

정규식으로 주소가 강원도로 시작하는 애들만 뽑으면 됨

# 작성한 코드

```sql
-- 코드를 입력하세요
SELECT factory_id, factory_name, address from food_factory where regexp_like(address, '^강원도') order by factory_id asc;
```
