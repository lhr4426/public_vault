# 문제 내용

문자열 `my_string`, `overwrite_string`과 정수 `s`가 주어집니다. 문자열 `my_string`의 인덱스 `s`부터 `overwrite_string`의 길이만큼을 문자열 `overwrite_string`으로 바꾼 문자열을 return 하는 solution 함수를 작성해 주세요.

## 입력

|my_string|overwrite_string|s|
|---|---|---|
|"He11oWor1d"|"lloWorl"|2|

## 출력

HelloWorld

# 문제 분석

my_string을 s개 출력하고 나머지를 overwrite_string을 출력한 뒤, my_string에서 남은 글자를 출력하면 됨

# 작성한 코드

```cpp
#include <string>
#include <vector>

using namespace std;

string solution(string my_string, string overwrite_string, int s) {
    string answer = my_string;
    
    for(int i = s ; i < s + overwrite_string.size(); i++) {
        answer[i] = overwrite_string[i-s];
    }
    return answer;
}
```

# 우수 코드 분석

```cpp
#include <string>
#include <vector>

using namespace std;

string solution(string my_string, string overwrite_string, int s) {
    string answer = "";
    answer = my_string.replace(s,overwrite_string.size(),overwrite_string);
    return answer;
}
```
